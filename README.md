# Space

Single Page Application

---

## Installation

1. Clone repository

```bash
https://github.com/ma-foyez/space.git
```

2. Go to the folder

```bash
cd space
```

3. Install Node packages

```bash
npm i
```

4. Run Local server with development for frontend change watching...

```bash
npm start
```

5. Open browser

```bash
http://localhost:3000
```
